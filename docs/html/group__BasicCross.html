Detailed Description

The cross product function calculates the cross product of two vectors and write the results into a destination vector.

Vector cross product function for f32X3.

Parameters
[out]dstpointer of the output vector
[in]src1pointer of the first input vector
[in]src2pointer of the second input vector
[in]countnumber of elements to be processed. Each element has a data type of riscv_vec_f32x3_t.
Returns
RISCV_VEC_OK

Example 

    #define count    3
    riscv_vec_f32x3_t src1[count] = {
        {0.1, 0.2, -0.1}, {-0.1, 0.3, 0.2}, {0.3, 0.1, -0.4}
    };
    riscv_vec_f32x3_t src2[count] = {
        {0.2, -0.3, 0.2}, {-0.5, 0.1, 0.3}, {-0.5, 0.1, -0.2}
    };
    riscv_vec_f32x3_t dst[count];
    riscv_vec_cross_f32x3(dst, src1, src2, count);
