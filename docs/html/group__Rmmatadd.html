Detailed Description

The row-major matrix addition function adds two elements from two source matrices and writes the results one-by-one in row-major order into a destination matrix.

Example

The code example with this function is as follows:
     #define row   2
     #define col   2
     float32_t src1[row * col] = {0.1, 0.4, -0.2, 0.1};
     float32_t src2[row * col] = {-0.2, -0.1, 0.3, 0.5};
     float32_t dst[row * col];
