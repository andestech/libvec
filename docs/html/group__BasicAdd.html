Detailed Description

Vector addition functions add a source vector to another source vector/scalar and write the results one-by-one into a destination vector.

)

Vector addition function for f32.

Parameters
[out]dstpointer of the output vector
[in]src1pointer of the first input vector
[in]src2pointer of the second input vector
[in]countnumber of elements to be processed
Returns
RISCV_VEC_OK

Example 

    #define count    3
    float32_t src1[count] = {0.1, 0.2, 0.3};
    float32_t src2[count] = {0.2, -0.3, 0.4};
    float32_t dst[count];
    riscv_vec_add_f32(dst, src1, src2, count);

◆ riscv_vec_add_f32x2()

int riscv_vec_add_f32x2 ( riscv_vec_f32x2_t *  dst,
riscv_vec_f32x2_t *  src1,
riscv_vec_f32x2_t *  src2,
uint32_t  count 
)

Vector addition function for f32x2.

Parameters
[out]dstpointer of the output vector
[in]src1pointer of the first input vector
[in]src2pointer of the second input vector
[in]countnumber of elements to be processed. Each element has a data type of riscv_vec_f32x2_t.
Returns
RISCV_VEC_OK

Example 

    #define count    3
    riscv_vec_f32x2_t src1[count] = {
        {0.1, 0.2}, {-0.1, 0.3}, {0.3, 0.1}
    };
    riscv_vec_f32x2_t src2[count] = {
        {0.2, -0.3}, {-0.5, 0.1}, {-0.5, 0.1}
    };
    riscv_vec_f32x2_t dst[count];
    riscv_vec_add_f32x2(dst, src1, src2, count);
 This example also serves as a reference for examples of f32x3 and f32x4 vector absolute functions.
