Detailed Description

These functions compute the distance between two vectors of float values: F32 and F16. Andes vector library supports algorithms for calculating the following distances:

  • Bray-Curtis distance
  • Canberra distance
  • Chebyshev distance
  • Cityblock (Manhattan) distance
  • Correlation distance
  • Cosine distance
  • Euclidean distance
  • Jensen-Shannon distance
  • Minkowski distance

Example

     #define order 3
     float32_t src1[size] = {1.0, 0.5, 0.4, -0.1};
     float32_t src2[size] = {0.40, -0.2, 0.24, 0.4};

     dist_out = riscv_vec_dist_minkowski_f32(src1, src2, order, size);
