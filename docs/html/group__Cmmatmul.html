Detailed Description

Column-major matrix multiplication functions compute the multiplication of two source matrices and write the results into a destination matrix in column-major order.

Column-major matrix multiplication function for MAT2X2.

Parameters
[out]dstpointer of the output matrix
[in]src1pointer of the first input matrix
[in]src2pointer of the second input matrix
[in]countnumber of elements to be processed. Each element has a data type of riscv_vec_cmmat2x2_f32_t.
Returns
RISCV_VEC_OK

Example 

The following example code presents the multiplication of four MAT2x2 matrices with this function.
     #define count 4
     riscv_vec_cmmat2x2_f32_t src1[count] = {...};
     riscv_vec_cmmat2x2_f32_t src2[count] = {...};
     riscv_vec_cmmat2x2_f32_t dst[count];
     riscv_vec_cmmat2x2_mul_f32(dst, src1, src2, count);
This example also serves as a reference for examples of MAT3x3 and MAT4x4
column-major matrix multiplication functions.
