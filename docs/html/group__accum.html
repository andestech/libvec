Detailed Description

Accumulation functions calculate the accumulate value of input vectors and return the accumulate results.

Andes vec library supports distinct accumulation functions for single-, half- and double-precision floating-point, Q31, Q15 ,Q7 and U8 data types. These functions are introduced in the subsections below.
