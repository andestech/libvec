Image median blur function for F32.

Parameters
[in]img_srcPoints to the input image structure
[out]img_dstPoints to the output image structure
[in]ksizeSize of the median kernel, it only supports for 3 and 5
Returns
RISCV_VEC_INPUT_PARAM_ERROR if the input ksize is not equal to 3 or 5, or the ksize is greater than and equal to img_src.height and img_src.width. RISCV_VEC_MALLOC_FAIL if allocation fails RISCV_VEC_SUCCESS if successful
