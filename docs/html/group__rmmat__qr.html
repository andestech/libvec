QR decomposition of a m x n floating point matrix with m >= n.

Parameters
[in]srcpoints to input matrix structure. The source matrix is modified by the function.
[in]thresholdnorm2 threshold.
[out]dstRpoints to output R matrix structure of dimension m x n
[out]dstQpoints to output Q matrix structure of dimension m x m
[out]taupoints to Householder scaling factors of dimension n
[in,out]pTmpApoints to a temporary vector of dimension m.
[in,out]pTmpBpoints to a temporary vector of dimension n.
[in]rownumber of rows in a matrix.
[in]colnumber of cols in a matrix.
Returns
RISCV_VEC_FAIL if the decomposition is not solved. RISCV_VEC_OK if the decomposition is solved.
